# SaaS Demo Backend

Backend API for a multi-tenant SaaS application built with Java 26 and Spring Boot.

This project focuses on authentication, authorization, tenant isolation, reusable backend patterns and business logic.

## Features

### Authentication & Security

* Auth0 authentication
* JWT authorization
* Role-based permissions
* Module-based permissions

### Multi-Tenant

* Tenant isolation
* Tenant switching
* User impersonation
* Tenant management

### Data Access

* Generic repositories
* Specification Pattern
* Filtering
* Pagination

### Infrastructure

* Global exception handling
* Scheduled tasks
* Unit tests
* Docker support

## Tech Stack

* Java 26
* Spring Boot
* Spring Security
* JPA / Hibernate
* PostgreSQL
* Docker
